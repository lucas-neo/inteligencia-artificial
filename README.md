# inteligencia-artificial

Repositório usado para organizar materiais da disciplina de Inteligência Artificial.

## Identificação

- Instituição: Unicesumar
- Curso: Engenharia de Software
- Professor: Munif Gebara

## Organização

O projeto está dividido em duas pastas principais:

- `Exercicios/`: reúne exercícios e materiais de apoio relacionados aos slides e atividades da disciplina.
- `Trabalhos/`: reúne entregas maiores, notebooks, bases de dados e registros das atividades.

## Estrutura atual

- `Exercicios/000`
- `Exercicios/002`
- `Exercicios/004`
- `Exercicios/006`
- `Trabalhos/001`
- `Trabalhos/003`

## Observações

- Cada pasta pode conter seu próprio `README.md` para explicar o conteúdo daquele diretório.
- `Trabalhos/001` contém um notebook com a atividade, a base `data.csv` e a identificação do grupo.
- `Trabalhos/003` contém um notebook, um arquivo CSV de apoio, uma imagem com a resposta e um `README.md` descritivo.
- Algumas pastas de exercícios podem conter apenas arquivos de apoio ou respostas curtas, dependendo da atividade proposta em aula.
